Live airspace monitoring

Autonomous Airspace Compliance for BVLOS Operations

Real-time 3D airspace intelligence, TFR monitoring, and weather-aware routing. All via a single, precise API built for autonomous flight systems.

View API Docs See it in action

Everything for compliant flight

From pre-flight planning to real-time position monitoring, NavGuard handles the complexity of 3D airspace compliance.

Pre-flight Airspace Scan

Query /scan to analyze complete flight corridors. Detects TFRs, weather restrictions, and no-fly zones across surface, 925hPa, and 850hPa altitude bands.

Real-time SSE Monitoring

Open a persistent stream to /zones/stream for live airspace updates. Redis-backed pub/sub distributes alerts across any number of server instances.

Flight Session Tracking

Start sessions with /flight/start, stream GPS positions to /flight/:id/position, and receive immediate safety assessments with weather and TFR alerts.

Contingency Validation

Update emergency landing points mid-flight via /flight/:id/contingency. Validates weather, regulatory restrictions, and safe airspace within 500m buffer zones.

TFR Sync & Alerting

AeroAPI integration pulls active TFRs every 10 minutes. PostGIS spatial queries detect new TFR activation in real-time during active flight sessions.

3D Weather Grid

Open-Meteo forecasts synced to a 0.25° grid across 12-hour horizons. Queries join on forecast_time closest to ETA, ensuring time-accurate weather assessment.

Plan a flight in one request

The /scan endpoint analyzes your entire flight corridor, returning safe airspace polygons, weather-restricted windows, and contingency validation.

  • GeoJSON output — Safe airspace, no-fly zones, and corridor buffers as standard geometries
  • Timeline analysis — Minute-by-minute weather forecast aligned to your flight path and ETA
  • Contingency checks — Validate emergency landing points against weather and regulatory data
cURL
# Scan a flight corridor from DFW to destination
curl -X POST https://api.navguard.io/api/scan   -H "Content-Type: application/json"   -H "x-api-key: ng_live_..."   -d '{
    "origin": {"lat": 32.7767, "lon": -96.7970},
    "destination": {"lat": 32.9, "lon": -96.5},
    "altitude_floor": 0,
    "altitude_ceiling": 400,
    "buffer_km": 10,
    "start_time": "2026-06-08T14:00:00Z",
    "thresholds": {
      "max_wind_mph": 25,
      "max_precip": 2,
      "min_visibility": 1000
    },
    "contingency_landing": {"lat": 32.85, "lon": -96.65}
  }'
